Privacy Policy

Last updated: March 28, 2026

1. Data We Collect

  • Account data: Email, name, avatar (optional)
  • Code and prompts: What you submit and what Codey generates
  • Usage data: Session timestamps, credits used, features accessed
  • Payment data: Processed by Stripe — we never store card numbers
  • GitHub tokens: Encrypted at rest, used only for repo access you authorize

2. How We Use Your Data

  • To provide the Codey service (code generation, analysis, builds)
  • To personalize your experience (memory system, preferences)
  • To process payments via Stripe
  • To send transactional emails (welcome, payment, low credits)

3. Data We Do NOT Collect or Share

  • We do not sell your data to third parties
  • We do not use your code for model training without explicit opt-in
  • We do not share code with third parties except for service delivery (LLM providers for generation)
  • User code is never sent to error tracking (Sentry)

5. Your Rights

  • Export: Full data export available anytime (code, memory, sessions, audit log)
  • Delete: Account deletion removes all data within 30 days
  • Edit: View and edit Codey's memory of you at /settings/memory
  • Opt out: Disable memory, notifications, and cross-project learning

6. Security

All data encrypted in transit (TLS 1.3) and at rest (AES-256). GitHub tokens encrypted with AES-256-GCM. Passwords hashed with bcrypt (cost factor 12). JWT tokens expire in 7 days.
