API Reference

All endpoints require JWT authentication unless noted. Pass token as Authorization: Bearer YOUR_TOKEN.

Base URL: https://codey-jc2r.onrender.com
POST /auth/signup
Create a new account. Returns JWT token and user object. No auth required.
ParameterType
emailstringrequired
passwordstringrequired
namestringoptional
curl -X POST https://codey-jc2r.onrender.com/auth/signup \ -H "Content-Type: application/json" \ -d '{"email":"you@example.com","password":"pass1234","name":"Your Name"}'
POST /sessions/prompt
Generate code from a natural language prompt. Returns generated code, credit cost, structural health report.
ParameterType
promptstringrequired
languagestringoptional (python, javascript, etc)
curl -X POST https://codey-jc2r.onrender.com/sessions/prompt \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"prompt":"Write a Python function that checks if a number is prime"}'
// Response { "session_id": "uuid", "estimated_credits": 1, "output": "```python\ndef is_prime(n):\n ...\n```", "lines_generated": 15, "status": "completed", "security_score": 1.0, "security_issues": [], "health": { "phase": "Excellent", "health_score": 0.85, "summary": "Code is structurally healthy." } }
POST /sessions/run
Execute code in an isolated sandbox. Returns stdout, stderr, exit code.
ParameterType
codestringrequired
languagestringoptional (default: python)
POST /analyze/code
Analyze code for structural health. Returns phase, ES score, coupling, cohesion, stress components, recommendations.
ParameterType
codestringrequired
filenamestringoptional
languagestringoptional
GET /users/me
Get current user profile, plan, credit balance.
GET /billing/plans
List all subscription plans with pricing and features.
POST /billing/subscribe
Subscribe to a plan. Returns Stripe Checkout URL for payment.
ParameterType
planstringrequired (starter, pro, team)
